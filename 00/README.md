**以实践为基础**

 doctest (<https://docs.python.org/3/library/doctest.html>) 是 Python 的一个标准库， 做测试用的。

结合本书的源码 （<https://github.com/fluentpython/example-code>）你可以在操作系统的控制台里键入 **下面的代码** 来验证书中几乎所有代码的正确性

```bash
python3 -m doctest example_script.py
```
